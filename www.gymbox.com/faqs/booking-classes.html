GYMBOX is famous for running the best and most creative studio timetable in the country. To secure a place on your favourite class we offer advanced booking through the Gymbox App.

Members can book studio classes online from 10:00am, 7 days in advance. The booking system will not allow a member to book onto classes that overlap each other.

Please note that to ensure full attendance, we operate a "3 strikes" booking system, whereby three no-shows in any 30 day period will result in a one-week advance booking restriction; please always ensure you cancel your class booking no later than one hour in advance of the class time, if you are no longer able to attend.

If you gain entry to the club without presenting your physical membership card this may result in an unrecorded attendance and a class booking strike will be applied, so you'll need to sign yourself in manually on the iPads located outside each studio - this will be using the last 4 digits of your phone number.

If a class is full you will be given the option to book onto a waiting list. Once you have been booked onto the waiting list you will receive an email to confirm. When a space becomes available you will be automatically moved into the class and notified via email up until an hour before the class start time.

Once on the waiting list you are committing to attend and could be auto booked into the class up to 1 hour before the booking start time. If you decide you can no longer attend please remove yourself from the waiting list as a strike will be applied in line with our cancellation policy. 　

Participants booked into class should swipe in the club at least 5 minutes before class start time, with participants expected to arrive into class before the class start time. If a participant arrives late, the booking is automatically removed from the class and opened up for other members at the class start time, if you do arrive late and there is a space still available, to avoid a booking ban please ensure you book back into the class otherwise a strike will be applied.
